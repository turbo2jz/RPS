## 1 项目说明

### 1.1 项目来由

SDUWH 2019-2020第一学期 **计算机网络** 实验四

### 1.2 项目要求

* 用熟悉的语言模拟实现 RIP 协议
* 支持多个路由器模拟
* 支持路由器学习过程日志输出
* 支持路由器路由信息输出（查看每个路由器的路由表）
* 支持模拟网络故障
* 支持模拟路由器故障
* 支持图形化界面（显示、交互）

[^详细要求]: 见Project/doc/实验4.pdf
